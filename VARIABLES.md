# Env Variables
This are the variables that should be set up for a production server

## SERVER
- SERVER_USERNAME
- SERVER_PASSWORD

## REDDIT

- REDDIT_USERNAME
- REDDIT_PASSWORD
- REDDIT_CLIENT_ID
- REDDIT_CLIENT_SECRET

## FIREBASE

- FIREBASE_API_KEY
- FIREBASE_AUTH_DOMAIN
- FIREBASE_DATABASE_URL
- FIREBASE_STORAGE_BUCKET
- FIREBASE_SERVICE_ACCOUNT
- FIREBASE_USER_EMAIL
- FIREBASE_USER_PASS
